# SheetFlow_AI_Website

## Project Overview

SheetFlow AI is a next-generation spreadsheet automation platform with an AI-native product website. This workspace contains all design systems, brand assets, content, 3D/motion references, and production files needed to build a world-class web experience.

## Workspace Structure

### 📋 Core Organization

- **01_Brand_Research** - Brand identity, positioning, typography, color systems, and competitive analysis
- **02_Design_Inspirations** - Visual references, screenshots, layouts, and UI pattern libraries
- **03_UI_Systems** - Component systems, glassmorphism patterns, spacing, buttons, dashboards
- **04_3D_Concepts** - Spline scenes, AI core visuals, workflow universe, particle systems
- **05_Motion_References** - GSAP animations, scroll storytelling, transitions, cursor effects
- **06_Content_System** - Copy library, CTAs, headlines, storytelling frameworks
- **07_Website_Architecture** - Sitemaps, UX flows, wireframes, interaction maps
- **08_Assets** - Logos, icons, textures, videos, mockups, exports
- **09_Spline_Scenes** - Interactive 3D Spline scenes for website integration
- **10_Final_Website** - Production website code, components, styles, deployment

### 📚 Documentation

- **docs/roadmap** - Project timeline and milestones
- **docs/implementation** - Technical implementation guides
- **docs/tech_stack** - Technology choices and justifications
- **docs/production_notes** - Production deployment and maintenance

### 💡 Resources

- **prompts** - AI prompts for design, copywriting, motion, and image generation
- All assets are organized by type for easy access and versioning

## Getting Started

1. Review `docs/roadmap.md` for project timeline
2. Check `docs/tech_stack.md` for technology decisions
3. Use `prompts/` folder for AI-assisted content creation
4. Reference `02_Design_Inspirations/` for style direction
5. Build components in `10_Final_Website/` once architecture is finalized

## Key Principles

- **AI-Native Design** - Leverage AI for rapid iteration and personalization
- **Motion-First** - Sophisticated animations with GSAP and Spline integration
- **Component-Driven** - Reusable, scalable UI systems
- **Content-Centric** - Compelling copywriting and visual storytelling
- **Production-Ready** - Built for performance, accessibility, and scalability
