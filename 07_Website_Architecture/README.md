# 07_Website_Architecture

## Purpose

Information architecture, user experience flows, and interaction design. This folder defines the structure, navigation, and user journey through the website.

## Contents

### sitemap/
- Complete website hierarchy
- Page list with URLs
- Navigation structure
- Taxonomy and categorization
- Page relationships and cross-links

### ux_flow/
- User journey maps
- Task flows for key conversions
- Navigation flows
- Signup and onboarding flows

### wireframes/
- Low-fidelity page layouts
- Component placement diagrams
- Grid and breakpoint layouts
- Mobile and desktop versions

### section_breakdowns/
- Hero section structure
- Features section layout
- Pricing section organization
- Testimonials and FAQ sections

### interaction_maps/
- Hover state specifications
- Click interaction targets
- Scroll trigger points
- Keyboard navigation paths

## Responsive Breakpoints

- Mobile: 320px - 640px
- Tablet: 641px - 1024px
- Desktop: 1025px - 1920px
- Large Desktop: 1921px+
