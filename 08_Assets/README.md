# 08_Assets

## Purpose

All visual assets organized by type. This is the master asset library for the SheetFlow AI website.

## Contents

### logos/
- Primary logo (SVG, PNG)
- Logo variations (light, dark, icon-only)
- Logo usage guidelines
- Minimum size requirements
- Clear space rules

### icons/
- Custom icon set (SVG)
- Icon library references
- Feature icons
- UI/navigation icons
- Social media icons

### textures/
- Background textures
- Noise overlays
- Gradient meshes
- Grid patterns
- Surface textures

### videos/
- Hero background videos
- Feature demo recordings
- Animation exports
- Product walkthroughs

### mockups/
- Device mockups (desktop, laptop, mobile, tablet)
- Browser mockups
- Dashboard previews
- UI in context shots

### exports/
- Final optimized assets for web
- WebP/AVIF image formats
- Compressed video files
- Icon sprites
- Font files

## Naming Convention

`[category]_[description]_[variant]_[size].[ext]`

Examples:
- `logo_primary_dark_2x.svg`
- `icon_spreadsheet_outlined_24.svg`
- `mockup_dashboard_desktop_hero.png`
- `texture_noise_subtle_bg.png`

## Optimization Requirements

- Images: WebP with PNG fallback
- Videos: MP4 (H.264) with WebM fallback
- Icons: Optimized SVG, no inline styles
- Max image size: 500KB for hero, 100KB for UI elements
- Use srcset for responsive images
