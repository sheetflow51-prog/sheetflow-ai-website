# Screenshots — Frame-grab archive

> Visual evidence of moments worth studying. Drop frames here when you see something that surprises you — a hero hold, a hover, a transition seam, a hand-tuned shadow.

## What earns a screenshot

A frame earns a place in this archive only if it shows *one specific quality* clearly:

- **Tension** — a moment where the layout almost breaks, then doesn't
- **Weight** — a typographic choice that feels physically heavy
- **Light** — a glow or gradient that reads as light, not color
- **Restraint** — empty space that's clearly chosen, not accidental
- **Surprise** — an interaction that exceeds expectation by ~one degree

A screenshot of "a nice hero" is not enough. The note must isolate *why* the frame matters.

## Naming convention

```
[YYYY-MM-DD]_[source]_[what-it-shows].png
```

Examples:
- `2026-05-06_stripe_hero-headline-fade.png`
- `2026-05-04_linear_glass-card-hover-state.png`
- `2026-05-02_vercel_globe-edge-glow.png`

## Annotation format

For each screenshot, add an entry below:

```
### filename.png
- **From**: stripe.com hero (2026-05-06)
- **What I saw**: The headline fades to ~55% white at the bottom edge, not a solid color stop.
- **Why it works**: The fade implies depth — the text appears to recede into the canvas rather than sit flat.
- **SheetFlow status**: Adopted (see `01_Brand_Research/typography/typography_system.md` → `text-fade`)
```
