# Moodboards

Moodboards are not Pinterest dumps. They are **curated direction documents**
that answer: "what does this surface need to feel like?"

Each board has the same structure:

- **Direction** — the one-sentence emotion to chase
- **References** — 6–10 chosen images (linked or saved to `_archive/`)
- **Adjectives** — the words we'll use when judging the result
- **Anti-references** — the things this board explicitly is not

## How to assemble a moodboard

1. Save 8–12 candidate references to `_inbox/moodboard-{slug}/`.
2. Cull to 6–10. Be ruthless. Eight strong > twenty mixed.
3. Write the **Direction** sentence first. Refuse to start until you can.
4. List adjectives last — they should fall out of the references.
5. Anti-references force discipline. Always include at least three.
