# Social Posts

Single-image and carousel posts for LinkedIn, X (Twitter), and IG.
Same brand temperature as the website — never lighter, never louder.

## Formats

| Platform  | Aspect | Px           | Use                          |
|-----------|--------|--------------|------------------------------|
| LinkedIn  | 1.91:1 | 1200 × 627   | Default share card           |
| LinkedIn  | 1:1    | 1200 × 1200  | Quote / announcement         |
| LinkedIn  | 4:5    | 1080 × 1350  | Editorial portrait           |
| X         | 16:9   | 1600 × 900   | Standard                     |
| X         | 1:1    | 1080 × 1080  | Quote                        |
| IG feed   | 4:5    | 1080 × 1350  | Default                      |
| IG story  | 9:16   | 1080 × 1920  | Story + reel cover           |

## Carousel rules

- 6–10 slides max. Past 10 nobody finishes.
- Cover slide: one sentence, no logo
- Body slides: one idea each
- Final slide: minimal CTA, never "follow for more"
- Slide-to-slide consistency: same temperature, same type weight

## Type system

- Headline: Söhne Halbfett or GT America Bold, 64–88pt at export size
- Body: Inter Regular, 32–40pt
- One line per thought. Wrap by meaning, not by width.
- Never reverse-out type on a photo without an atmospheric dim layer

## What ships

- Quote posts from founder/team
- Build-in-public progress (rare — only on milestones worth showing)
- Behind-the-scenes glimpses from
  [10_Case_Study_Assets](../10_Case_Study_Assets/)
- Product moments cropped from
  [12_Final_Exports/stills](../12_Final_Exports/)

## What never ships

- Screenshots with arrows/circles overlaid
- "Day N of building" counter graphics
- Memes (unless explicitly approved)
- Carousels with >10 slides
- Posts using stock photography

## Workflow

1. Pick reference still or generate via MJ
2. Drop into Figma `social.fig` master template
3. Compose with type
4. Export as `social-{platform}-{slug}-v{n}-{YYYYMMDD}.png` to `_inbox/`
5. Run through `13_Asset_Review/` before publishing
