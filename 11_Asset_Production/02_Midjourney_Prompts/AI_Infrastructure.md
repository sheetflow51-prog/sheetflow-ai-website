landing pages, technical case studies, and developer-facing surfaces.

## What infrastructure imagery is NOT

- Stock photo "data center with green cables"
- "Glowing matrix of servers"
- "Hands typing code on multiple monitors"
- "Fingerprint hologram on circuit board"
- Any image with green binary text overlay

If a prompt could be on a McAfee homepage from 2008, reject.

## When to use these

- `/engineering` page hero
- Architecture/technical case studies
- Investor/decks where "this is real infrastructure" is the message
