the surface around the type** — actual type is set in Figma.

## Typography pairing recommendations

| Surface            | Display face       | Body face          | Notes                      |
|--------------------|--------------------|---------------------|----------------------------|
| Posters / hero     | GT America Mono    | Inter              | Confident, geometric       |
| Editorial          | Söhne Breit        | Söhne              | Magazine-grade gravitas    |
| Product / dashboard| Inter Display      | Inter              | Cohesive system            |
| Headlines (display)| Custom variable    | Inter              | Variable axis = motion lever |

## Type rules over generated backgrounds

1. **One type weight per composition.** Bold OR light, never both.
2. **Tracking on display sizes**: -0.02em to -0.03em. Caps: +0.04em.
3. **Type sits in the negative space**, never on the subject.
4. **Color: white at 92% opacity**, never pure white. (Pure white =
   harsh on cinematic backgrounds.)
5. **Drop shadows are forbidden.** If type doesn't read, the background
   is too busy — fix the background.
