# Cinematic Dashboard Prompts

Inspiration for product surfaces. Note: we generate these for **direction**,
not for use. Real product UI is built in code, not Midjourney.

---

## dash-glass-v1

```
medium shot of layered translucent frosted glass panels at staggered
depths against a deep ink background, each panel containing soft abstract
data visualizations using only matte ceramic forms and warm amber accent
lines, single key light from above cool 6000K, depth blur on background
panels, 85mm portrait lens, Apple Vision OS aesthetic, no labels, no
numbers, no text
--ar 16:9 --style raw --stylize 200 --v 6
--no neon, rgb, glow, ui text, numbers, words, holographic grid
```
