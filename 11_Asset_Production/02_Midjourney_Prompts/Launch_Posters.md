# Launch Poster Prompts

Key art for major SheetFlow moments. Posters are designed to read at both
a phone screenshot and a billboard.

---

## poster-launch-v1 — The announcement

```
vertical poster composition of a single matte ceramic geometric core
suspended in deep ink-black volumetric space, lit by cool sapphire key
from above and warm amber rim from behind, profound negative space in
upper third for typography overlay, Roger Deakins atmosphere, Apple
keynote restraint, editorial poster proportions
--ar 2:3 --style raw --stylize 220 --v 6
--no neon, rgb, glow, holographic, text, words, sparkle
```

After generation, type is added in Figma per
[Typography Compositions](Typography_Compositions.md).

---

## poster-product-v1 — The product hero

```
vertical poster shot of a brushed graphite device floating on subtle
```

## Poster delivery format

| File                                   | Format               |
|----------------------------------------|----------------------|
| Source image                           | Midjourney upscale, PNG, max quality |
| Layout file                            | Figma, type overlaid in `12_Final_Exports/posters/{slug}-v{n}.fig` |
| Print-ready                            | PDF, 300 DPI, 1.5x bleed             |
| Social-ready                           | PNG 1080×1620, sRGB                  |
| Story-ready                            | PNG 1080×1920, sRGB                  |
