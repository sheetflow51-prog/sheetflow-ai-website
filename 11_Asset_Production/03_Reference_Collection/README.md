# Reference Collection

The library we steal taste from. Not assets we ship — fuel for assets we make.

Every reference category answers four questions:

1. **What to learn** — the principle worth absorbing.
2. **What to avoid** — the trap that ruins it.
3. **What fits SheetFlow** — the angle we use.
4. **Cinematic observations** — what a DP would notice.

## Intake protocol

When you save a reference:

1. Drop the source file in `_inbox/references/{category}/`.
2. Rename to `{source}_{subject}_{YYYYMMDD}.{ext}`.
3. Add one line to the relevant category file with: source, why it's here,
   one cinematic observation.
4. Run `scripts/organize-inbox.ps1` weekly to fold inbox into archive.

Never paste links without a note. A naked link is dead weight in a year.
