## Copy Guidelines

### Tone & Voice
- **Professional** - Credible and knowledgeable
- **Approachable** - Accessible, not overly technical
- **Benefit-Focused** - Answer "what's in it for me?"
- **Action-Oriented** - Clear next steps
- **Inclusive** - Uses "we/you" language

### Structure
- **Headlines** - Attention-grabbing, benefit-driven
- **Body Copy** - Scannable, benefit-first, feature-second
- **CTAs** - Action verb, clear outcome
- **Social Proof** - Specific metrics and testimonials

### Writing Best Practices

1. **Front-load benefits** - Lead with value, not features
2. **Use active voice** - "Automate your workflows" vs "Workflows can be automated"
3. **Be specific** - "Save 10 hours/week" vs "Save time"
4. **Address pain points** - Acknowledge customer problems
5. **Provide proof** - Include numbers, testimonials, case studies
6. **Create urgency** - Limited offers, scarcity language
7. **Make it scannable** - Short paragraphs, bullet points, bold keywords

## Localization Prep

- Keep copy in markdown for easy translation
- Avoid idioms and culture-specific references
- Use simple sentence structures
- Include context notes for translators
- Plan for text expansion in German/French
