## Prompt Format

Each prompt file should include:

```markdown
## [Prompt Name]

**Purpose:** What this prompt accomplishes
**Model:** Claude / GPT-4 / Midjourney / etc.
**Context:** Background info to include

---

[PROMPT TEXT]

---

**Example Output:** Optional example of good output
**Variations:** Alternative versions of the prompt
**Notes:** Tips for getting best results
```

## Best Practices

1. **Be specific** - Include style, tone, constraints
2. **Provide context** - Share relevant background
3. **Iterate** - Save prompt variations that work
4. **Document results** - Note what worked and why
5. **Version control** - Date significant prompt updates
