# 02_Design_Inspirations

## Purpose

Visual reference library and design inspiration for UI patterns, layouts, and interactions. This folder helps maintain consistency and spark creative ideas throughout the design process.

## Contents

### screenshots/
- Website screenshots from inspiration sources
- Mobile and desktop viewport captures
- Detailed close-ups of specific patterns
- Annotated observations about design choices

### references/
- Design articles and case studies
- Figma community links
- Dribbble collections
- Behance projects
- Design system documentation from industry leaders

### layouts/
- Hero section variations
- Landing page grid patterns
- Feature section layouts
- Dashboard layouts
- Footer and navigation patterns

### hero_sections/
- Hero with video background
- Hero with 3D elements
- Hero with animated text
- Hero with scroll trigger animations
- Hero with gradient backgrounds

### ui_patterns/
- Button states and variations
- Form input patterns
- Card designs and layouts
- Modal and dialog patterns
- Navigation patterns
- Loading and progress states
- Hover and interaction effects

## How to Use

1. Collect inspiring designs as you browse
2. Organize by pattern type or visual style
3. Take screenshots with brief notes
4. Document what makes each design effective
5. Reference when designing new components

## Organization Tips

- Use consistent naming: `[type]_[style]_[source].png`
- Add descriptive filenames for easy searching
- Create a `NOTES.md` in each subfolder for observations
- Tag screenshots with design principles used
